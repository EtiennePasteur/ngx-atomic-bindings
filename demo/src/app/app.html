{{ title() }}

Dynamic component rendering with type-safe bindings using Angular signals

Example 1: Mixed Dynamic & Fixed Props

Combines dynamic props (event handler) with fixed props (variant). Click count: {{ clickCount() }}

// Mix dynamic and fixed props:
<ng-container
  [atomicBindingsProps]="dynamicButtonProps()"
  [atomicBindings]="DemoButtonComponent"
  [atomicBindingsFixed]="{ variant: 'success' }">
</ng-container>

clickCount = signal(0);

dynamicButtonProps = signal({
  label: `Button`,
  clicked: () => {
    this.clickCount.update(c => c + 1);
  },
});

Example 2: Atomic Design Pattern (Molecular + Atomic)

The card (molecular) uses the button (atomic) internally through the directive. This demonstrates how molecular components can expose atomic component props dynamically!

// Card component (Molecular) template:
<div class="card">
  ...
  <div class="card-footer">
    <ng-container
      [atomicBindings]="DemoButtonComponent"
      [atomicBindingsProps]="actionButtonProps()">
    </ng-container>
  </div>
</div>

// Card accepts button props dynamically:
actionButtonProps = input<ComponentProps<DemoButtonComponent>>();

// Usage:
cardProps = signal({
  title: 'Dynamic Card (Molecular)',
  content: '...',
  actionButtonProps: {
    label: 'Learn More',
    variant: 'primary',
    clicked: () => alert('Clicked!'),
  },
});
